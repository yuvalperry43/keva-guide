# כלכלה בקבע — המדריך המלא

כדי להכיר מה שמגיע לכם ולמקסם את כל הזכויות שלכם.

המדריך הזה נכתב מניסיון אישי, לא מדפי מידע רשמיים. המטרה היא פשוט לחסוך לכם את הזמן ואת הטעויות שכבר עשיתי.

## איפה להתחיל?

אם אתם לא יודעים מאיפה להתחיל — התחילו עם [מועדון חבר](הטבות-וזכויות/מועדון-חבר.md) ו[בנק אוצר החייל](הטבות-וזכויות/בנק-אוצר-החייל.md). שם נמצא הכסף הכי קל לתפוס.
